Opis szczegółowy

template<typename Type, int Dimension>
class Matrix< Type, Dimension >

Szablon definiuje pojęcie Matrix<Type,Dimension>, gdzie:

  • Type oznacza typ liczbowy jaki przechowuje macierz
  • Dimension określa wymiary macierzy kwadratowej
