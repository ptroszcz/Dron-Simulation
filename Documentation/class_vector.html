Opis szczegółowy

template<typename Type, int Dimension>
class Vector< Type, Dimension >

Szablon definiuje pojęcie vector<Type,Dimension>, gdzie:

  • Type oznacza typ liczbowy jakim są wyrażone współrzednę wektora
  • Dimension określa wymiary wektora

Konstruktor szablonu vector<> ustawiająca pierwsze 3 wartosci wspolrzednych na zadane 3 wartosci dla wektora co najmniej o trzech wspolrzednych. Reszte wspolrzenych ustawia na zero. Jezeli zostanie podany wektor o mniej niz 3 wspolrzednych to wszystkie jego wspolrzedne zostana ustawione na zera.

Parametry
[in]x- pierwsza współrzędna wektora
[in]y- druga współrzędna wektora
[in]z- trzecia współrzędna wektora
